Technical Flow Breakdown:

  1. Trigger: A webhook in HubSpot fires on "Deal Won", sending customer and deal data to a secure endpoint.
  2. Gateway & Orchestration: Azure API Management receives the payload, authenticating and routing it to an Azure Logic App. This orchestrates the entire workflow.
  3. ERP Sync: The Logic App creates a new customer and project in the ERP via its REST API, transforming HubSpot data to match the ERP schema.
  4. Payment Initiation: Upon successful ERP entry, the Logic App calls the Stripe API to generate an invoice and a secure payment link.
  5. Financial Reconciliation: A second webhook from Stripe (on 'payment_successful') triggers another Logic App function to create a corresponding sales invoice in Xero.
  6. Notifications: The process concludes by posting real-time status updates to a dedicated Microsoft Teams channel for stakeholder visibility.
